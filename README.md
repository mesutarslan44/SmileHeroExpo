# SmileHeroExpo

SmileHeroExpo is a gamified oral care app that helps users build a consistent tooth-brushing habit.
It combines health education, playful motivation, and mobile feedback loops in a kid-friendly experience.

## What It Does

- Encourages regular brushing habits
- Uses gamified interaction patterns
- Supports habit building with reminders and engagement
- Turns oral care into a more motivating daily routine

## Tech Stack

- React Native
- Expo
- JavaScript
- Expo Notifications

## Project Goal

This project aims to make oral care more engaging through product design and positive reinforcement.
The focus is on turning a health routine into something users actually want to return to.

## Getting Started

```bash
npm install
npx expo start
```
